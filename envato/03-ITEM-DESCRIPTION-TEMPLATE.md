## Item Title

TakaTrack - React Native Expo Personal Finance App (iOS + Android)

## Short Description

Offline-first personal finance mobile app source code built with React Native and Expo. Includes income/expense tracking, category management, budget planning, analytics charts, export (CSV/PDF), and multilingual support (English/Bengali).

## Long Description

TakaTrack is a production-ready mobile finance tracker source code for Android and iOS. It is designed for quick customization and fast launch.

Main highlights:

- React Native + Expo architecture
- Local SQLite database with offline-first behavior
- Auth flow with optional Supabase integration
- Transactions, categories, and budget modules
- Analytics dashboards (daily/weekly/monthly)
- CSV and PDF export support
- English and Bengali language support
- Clean folder structure and modular services

What you get:

- Full source code
- Documentation (`README.md` + `docs/`)
- Setup and build instructions
- Support policy and changelog starter files
