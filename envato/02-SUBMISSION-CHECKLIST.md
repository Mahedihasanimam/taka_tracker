# Envato Submission Checklist

## Files and Packaging

- [ ] Project builds and runs with documented steps.
- [ ] `node_modules` is not included in upload zip.
- [ ] `.env`, keys, private files are not included in upload zip.
- [ ] Documentation is included (`README.md`, `docs/`, and `envato/`).
- [ ] License file is included (`LICENSE`).
- [ ] Third-party asset/dependency credits are included (`envato/07-CREDITS-AND-LICENSES.md`).

## Product Quality

- [ ] UI has no broken screens.
- [ ] Core features work: auth, transactions, budgets, export, profile.
- [ ] No console crashes in normal flows.
- [ ] Lint warnings reduced or explained.
- [ ] App icon/splash/branding are consistent.

## Marketplace Content

- [ ] Correct category selected (recommended: CodeCanyon mobile app source code).
- [ ] Item title finalized.
- [ ] Short description and long description prepared.
- [ ] 15 tags prepared.
- [ ] Thumbnail and preview images prepared.
- [ ] Demo video prepared (optional but strongly recommended).

## Commercial Readiness

- [ ] Support policy and response time defined.
- [ ] Changelog file initialized (`envato/06-CHANGELOG.md`).
- [ ] Pricing selected (see `envato/04-PRICING-GUIDE.md`).
- [ ] You tested a fresh install from the packaged zip.
