# What You Still Must Do (Manual Tasks)

These tasks cannot be fully automated from this repo and must be done by you in Envato and store accounts.

## Envato Account Tasks

1. Complete Envato author profile (identity verification, tax, payout).
2. Choose marketplace/category (recommended: CodeCanyon mobile app source).
3. Create item page using `envato/03-ITEM-DESCRIPTION-TEMPLATE.md`.
4. Upload preview images, thumbnail, and demo media.
5. Set initial price (see `envato/04-PRICING-GUIDE.md`).
6. Submit item and handle review feedback.

## Product Branding Tasks

1. Replace placeholder bundle/package IDs if needed:
   - iOS `bundleIdentifier`
   - Android `package`
2. Replace app name/slug if you want final brand consistency.
3. Ensure icon/splash and screenshots match final brand.

## Supabase / Backend Tasks

1. Decide if buyers will use Supabase or local-only mode.
2. If Supabase is required, provide SQL/setup steps in docs.
3. Replace demo Supabase keys with your own production-safe setup guidance.

## Sales Readiness Tasks

1. Create short demo video (60 to 120 seconds).
2. Prepare FAQ and support email.
3. Test install on clean machine from packaged zip.
