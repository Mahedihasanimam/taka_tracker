# Marketplace Choice (Important)

This product is a React Native + Expo mobile source code app.

- ThemeForest is mainly for website themes/templates.
- This product type is a better fit for CodeCanyon (Mobile App Templates / Full App Source Code category).

If you still submit to ThemeForest, rejection risk is high because this is not a website theme.

Recommended path:

1. Submit this mobile source code to CodeCanyon.
2. If needed, later create a website landing page template version for ThemeForest separately.
